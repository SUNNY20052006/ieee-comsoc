# /public/logos

Place the following logo files in this directory:

- `comsoc-logo.png` — IEEE Communication Society logo (transparent background, recommended 200x60px)
- `iem-logo.png` — IEM logo (transparent background, recommended 200x60px)

Both files should have transparent backgrounds for the best appearance on the dark hero section and the light-mode navbar.

## Sources
- ComSoc logo: https://www.comsoc.org/about/branding
- IEM logo: Available from the Institute's official resources

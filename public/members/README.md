## Specs
- **Format**: JPG or PNG
- **Dimensions**: 200×200px minimum (square crop recommended)
- **Max file size**: 200KB per image (use tinypng.com to compress)

If a photo is missing, the card will gracefully fall back to displaying the member's initials on a colored background.
